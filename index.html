Self-Stabilizing Spoon

Empowering People with Parkinson’s Disease

What is it?

The Self-Stabilizing Spoon is a smart assistive device designed to help people with Parkinson’s disease eat more comfortably. It uses motion sensors and motors to detect hand tremors and stabilize the spoon in real-time.

Why is it Needed?

Parkinson’s disease causes hand tremors that make eating difficult. This device enhances independence, reduces frustration, and brings dignity back to mealtime.

Key Features

  • Built-in gyroscope and accelerometer
  • Real-time tremor compensation using motors
  • Rechargeable battery with long life
  • Interchangeable heads (spoon, fork)
  • Compact and ergonomic design

Electronic Components Used

  • Arduino Nano or Uno
  • MPU6050 (Accelerometer + Gyroscope)
  • 2 x Servo Motors (for X and Y axis stabilization)
  • Li-ion Battery + Charging Module
  • Switch / Button (to turn on/off)
  • Wires and Mounting Hardware
  • 3D-Printed or Lightweight Spoon Mount

Sample Arduino Code

#include <Wire.h>
#include <Servo.h>
#include <MPU6050.h>

MPU6050 mpu;
Servo servoX;
Servo servoY;

void setup() {
  Wire.begin();
  Serial.begin(9600);
  mpu.initialize();
  servoX.attach(9);
  servoY.attach(10);
}

void loop() {
  int16_t ax, ay, az, gx, gy, gz;
  mpu.getMotion6(&ax, &ay, &az, &gx, &gy, &gz);

  int angleX = map(gx, -17000, 17000, 0, 180);
  int angleY = map(gy, -17000, 17000, 0, 180);

  angleX = constrain(angleX, 60, 120);
  angleY = constrain(angleY, 60, 120);

  servoX.write(angleX);
  servoY.write(angleY);

  delay(20);
}

How It Works

The MPU6050 senses angular changes due to tremors. Arduino processes these inputs and sends corrective signals to two servo motors, keeping the spoon level despite unintended hand movements.
